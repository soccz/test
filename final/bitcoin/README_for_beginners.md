# 누구나 이해하는 비트코인 예측 AI 설명서 📖

이 문서는 코드를 전혀 모르는 분들도 **"도대체 이 프로그램이 어떻게 비트코인 가격을 맞추겠다는 거야?"** 라는 궁금증을 해소하실 수 있도록 아주 쉽게 설명합니다.

---

## 1. 어떤 데이터를 쓰나요? (공통 기준) 📅
두 프로그램 모두 **"지난 200일간의 하루하루 가격(일봉)"**을 살펴보고, **"바로 내일의 가격"**을 예측합니다.

*   **관찰 기간:** 최근 200일 (약 6개월 반)
*   **예측 대상:** 내일의 비트코인 가격 (Next Day Price)

---

## 2. 사용된 용어 설명 (이게 다 뭔가요?) 🧐

코드에 나오는 영어 단어들이 실제로는 어떤 의미이고, 어떻게 비유할 수 있는지 정리했습니다.

| 전문 용어 (Variable) | 쉬운 설명 (Meaning) | 비유 (Metaphor) |
| :--- | :--- | :--- |
| **Open / Close**<br>(시가 / 종가) | 하루 장이 시작할 때 가격과<br>끝날 때 가격 | 가게 문 열 때 계란값,<br>문 닫을 때 계란값 |
| **Volume**<br>(거래량) | 하루 동안 거래된 비트코인 총 개수 | 그날 가게에 손님이<br>얼마나 북적였는지 (인기 척도) |
| **MA (Moving Average)**<br>(이동평균선) | "최근 5일 평균", "최근 20일 평균" 등<br>가격의 흐름을 부드럽게 본 것 | 하루하루 기분 말고,<br>**"요즘 대체로 기분이 어때?"** (평균 상태) |
| **Epoch**<br>(에포크) | 인공지능이 데이터를 반복해서<br>공부한 횟수 | 시험범위 문제집을<br>**N번 복습**했다 |
| **Linear Regression**<br>(선형 회귀) | 데이터를 가장 잘 설명하는<br>직선 하나 긋기 | "지난달에도 올랐으니<br>다음달도 이만큼 오르겠지?" (단순 추세) |
| **Deep Learning**<br>(딥러닝) | 뇌세포처럼 복잡하게 연결된<br>신경망으로 패턴 찾기 | "차트 모양이 M자니까<br>떨어질 확률 80%야" (고수 패턴 분석) |

---

## 3. 두 가지 예측 방법 🤖

### � 방법 1: 단순 추세선 (`bitcoin_basic.py`)
> **"전체적인 흐름(분위기)만 봅니다."**

*   지난 200일 동안 가격이 우상향(올라감)했으면 "내일도 오를 거야"라고 하고, 우하향(내려감)했으면 "내일은 내릴 거야"라고 단순하게 선을 그어 예측합니다.
*   하루하루의 등락보다는 **큰 방향성**을 볼 때 좋습니다.

### 🔴 방법 2: 딥러닝 분석 (`bitcoin_deep.py`)
> **"최근 패턴을 공부해서 내일을 맞춥니다."**

*   지난 200일간의 가격뿐만 아니라 거래량, 5일/20일 평균선 등 다양한 힌트를 같이 봅니다.
*   **"최근 10일간 이런 모양이었을 때 다음날 어떻게 되더라?"** 라는 패턴을 학습해서 내일 가격을 찍습니다.
*   단순 직선보다 더 구체적인 값을 내놓지만, 틀릴 수도 있습니다.

터미널에 아래 명령어를 입력하고 엔터를 치세요.

python bitcoin/bitcoin_basic.py

**딥러닝 모델 실행:**
